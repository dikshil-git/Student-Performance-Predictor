# 📊 Student Performance Predictor

## 📌 Overview
This project predicts a student's final score based on:
- Hours studied
- Previous scores

It uses a simple Machine Learning model (Linear Regression).

---

## ⚙️ Tech Stack
- Python
- Pandas
- Scikit-learn
- Matplotlib

---

## 📊 Features
- Data preprocessing
- Model training
- Prediction system
- Data visualization

---

## ▶️ How to Run
1. Install dependencies:
   pip install pandas scikit-learn matplotlib

2. Run the notebook:
   main.ipynb
